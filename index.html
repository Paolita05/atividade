✅Regra da Cascata

A cascata é o mecanismo do CSS que define qual estilo será aplicado quando há conflito. Ordem de prioridade:

  • Importância: regras com !important têm prioridade máxima.
  • Especificidade: ID > Classe > Tag.
  • Ordem: a última regra escrita no código vence.

🖥️ Especificidade

  • É o "peso" de cada seletor.
  • Tag (elemento) → peso baixo (ex.: p, h1).
  • Classe, pseudo-classes, atributos → peso médio (ex.: .caixa, :hover).
  • ID → peso alto (ex.: #menu).
  • Estilo inline → prioridade máxima (ex.: style="color:red").

🔗Combinação de seletores

  • Descendente (espaço): seleciona qualquer elemento dentro de outro, mesmo que não seja filho direto.
  • Filho direto (>): seleciona apenas o elemento que está logo dentro do pai.
  • Adjacente (+): seleciona o elemento que aparece imediatamente depois de outro.

✅Pseudo-classes

  • :first-child → seleciona o primeiro filho de um elemento.
  • :last-child → seleciona o último filho de um elemento.
  • :nth-child(n) → seleciona o filho na posição n.
  • :nth-of-type(n) → seleciona o n-ésimo filho de um determinado tipo (por exemplo, o 3º

⸜(｡˃ ᵕ ˂ )⸝ Diferença entre Display Inline e Block Block:

  • Ocupa 100% da largura.
  • Quebra linha antes e depois.
  • Aceita width, height, margin e padding.
  • Exemplos: div, p, h1-h6, section.
  • Inline:
  • Ocupa só o espaço do conteúdo.
  • Não quebra linha.
  • Não aceita width e height (exceto img).
  • Exemplos: span, a, strong, em.

🔎Margin, Padding e Border

  • Margin: espaço externo (fora da borda).
  • Padding: espaço interno (entre conteúdo e borda).
  • Border: a borda do elemento (cor, estilo e espessura).

🖊Shorthand (formas de escrita):

  • 1 valor → todos os lados iguais.
  • 2 valores → primeiro vertical (top/bottom), segundo horizontal (left/right).
  • 3 valores → top, horizontal (left/right), bottom.
  • 4 valores → top, right, bottom, left (sentido horário).

₍ᐢ. .ᐢ₎ Box-sizing

  • Define como o navegador calcula largura e altura.
  • box-sizing: border-box; → o tamanho do elemento considera conteúdo + padding + borda, evitando que ultrapasse a largura definida.
